# Design Direction

## Core promise

A polished, ever-expanding choice-driven adventure where the player feels the weight of a long dangerous road, an unstable fellowship, and a relic that can help or corrupt.

## Legal/content boundary

The core repository uses original names, scenes, prose, and systems. The intended flavor is mythic high fantasy, not copied franchise material.

A future `lore-packs/` system can support private adaptation layers, but the public core should remain original and safe to share.

## Feature pillars

1. **Branching scenes with memory**
   Choices alter stats, flags, relationships, and future scene availability.

2. **Travel as gameplay**
   Routes should have terrain, danger, supplies, discoveries, and clocks.

3. **Fellowship pressure**
   Companions should disagree, fear, trust, betray, or save the player.

4. **Relic temptation**
   Power should solve problems while creating long-term shadow cost.

5. **Replayable myth**
   Multiple openings, hidden roads, endings, oaths, and consequences.
